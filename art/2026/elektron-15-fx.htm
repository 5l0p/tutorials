Elektron Synthesis - Part 15: Machinedrum and Monomachine Effects in Pd

Both the Machinedrum and Monomachine include built-in effects processing — not just simple reverb/delay, but a structured signal routing architecture that shapes the overall mix. Understanding how the effects work helps you replicate the Elektron "sound" in a Pd implementation: tight per-track compression on the MD, the MNM's per-voice FX sends, and the global effects bus shared across all tracks.

Machinedrum Effects Architecture

The MD has three effect stages:

  1. Per-track dynamics: each track has an independent LFO and volume control. No per-track effects.
  2. Global delay: a stereo delay with ping-pong mode and high/low frequency filtering in the feedback path.
  3. Global reverb: a plate-style reverb fed from a per-track send level.

Machinedrum Global Delay

; MD stereo delay with filtered feedback in Pd:

; Per-track send: voice_output × (send/127) → delay input bus

; Left channel delay:
[delwrite~ md_dl 3000]    ; 3000ms max buffer

[delread~ md_dl $timeL]
|
[hip~ $hpfreq]            ; highpass in feedback path (removes sub rumble)
|
[lop~ $lpfreq]            ; lowpass in feedback path (darkens over time)
|
[*~ $feedback]            ; feedback level
|
[+~] ← delay input bus   ; add to incoming signal
|
[delwrite~ md_dl 3000]    ; (circular: also goes back into delwrite)

[delread~ md_dl $timeL]
|
[*~ $mixlvl]              → left output

; Right channel: same structure with $timeR
; Ping-pong: left delay's output feeds right delay input and vice versa:
; left_out → right_delwrite_input
; right_out → left_delwrite_input
; (instead of each feeding back to itself)

Machinedrum Global Reverb

The MD reverb is a plate-style reverb — dense, bright, with a characteristic shimmer that works well on percussion. In Pd, approximate with [rev3~] tuned to plate characteristics, or use a Schroeder/FDN reverb design: 

; MD-style plate reverb using [rev3~]:
; rev3~ takes: room-size ms, reverb-time ms, damping
; Plate-like settings: small room size, long reverb time, low damping

[+~] all voice sends (each × their $send level)
|
[rev3~ 8 $verbTime 0.3]   ; 8ms room, long tail, minimal damping
|                     |
wet L               wet R
|                     |
[*~ $mixlvl]         [*~ $mixlvl]
→ added to main stereo output

Monomachine Effects Architecture

The MNM has a more sophisticated effects routing:

  • Per-voice: each track has EQ, distortion, and a send level to the global effects bus
  • Global delay: stereo delay with modulation (chorus-delay)
  • Global reverb: more complex room/hall model than the MD
  • Global chorus/flanger: shared modulation effect with per-track send

MNM Per-Voice EQ

ParameterFunction
BASE (mid frequency)Peaking EQ band centre frequency (80–12000 Hz)
WIDTHEQ bandwidth (Q factor)
GAINBand boost/cut in dB (±18 dB)

A single parametric peaking EQ band per voice. Implement in Pd with a biquad peaking filter: 

; Peaking EQ biquad coefficients (Audio EQ Cookbook):
; A = sqrt(10^(dBgain/20))
; w0 = 2π × fc / Fs
; α = sin(w0) / (2 × Q/BW)
;
; b0 =  1 + α*A
; b1 = -2*cos(w0)
; b2 =  1 - α*A
; a0 =  1 + α/A
; a1 = -2*cos(w0)
; a2 =  1 - α/A
;
; normalised (divide by a0):
; pass to [biquad~]:
; [biquad~ b1/a0 b2/a0 a1/a0 a2/a0 b0/a0]
; in Pd biquad~ format: [biquad~ a1 a2 b0 b1 b2]
; (Pd has its own coefficient ordering: see Pd manual)

MNM Per-Voice Distortion

Each MNM voice has a built-in distortion (soft clip). This is the same tanh waveshaper used in the drum machine articles, applied after the filter and before the FX send: 

VCA_output
|
[expr~ tanh($v1 * $drive) / tanh($drive)]   ; normalised tanh waveshaper
|
EQ
|
FX send bus

MNM Global Delay with Modulation

ParameterFunction
TIMEDelay time (ms, tempo-syncable)
FEEDBACKFeedback level
MOD DEPTHLFO modulation of delay time (chorus effect)
MOD SPEEDLFO rate for delay time modulation
WIDTHStereo spread (L/R delay time offset)
; Modulated stereo delay (chorus-delay):

; LFO for delay time modulation:
[phasor~ $modSpeed]    ; LFO oscillator
|
[*~ 2][-~ 1]           ; -1 to +1
|
[*~ $modDepth]         ; depth in ms
|
[+ $time]              ; add to base delay time → modulated time signal
|
[delread~ mnm_d1]      ; variable delay read using modulated time

; Stereo: right channel uses opposite LFO phase:
[phasor~ $modSpeed]
|
[+~ 0.5][wrap~]        ; shift phase 180° for stereo opposition
|
[*~ 2][-~ 1]
|
[*~ $modDepth][+ $time + $width/2]
→ [delread~ mnm_d2]    ; right channel

MNM Global Chorus/Flanger

ParameterFunction
DEPTHModulation depth (ms for chorus, cents for flanger)
SPEEDLFO rate
FEEDBACKFeedback (0=chorus, positive=flanger, negative=reverse flanger)
MIXWet level
; Chorus: DEPTH large (10–30ms), FEEDBACK near 0
; Flanger: DEPTH small (0–10ms), FEEDBACK high (0.5–0.9)

; Implementation:
FX_send_bus
|
[delwrite~ chorus 50]

[phasor~ $speed]
|
[*~ 2][-~ 1]
|
[*~ $depth]              ; LFO in ms
|
[+ $baseDelay]           ; base delay: chorus ~15ms, flanger ~2ms
|
[delread~ chorus]        ; variable read
|
[*~ $feedback]
→ back to delwrite input  ; creates flanger comb filter at high feedback

[delread~ chorus]
|
[*~ mix]
|
[+~] ← dry × (1-mix)
→ output

Global Reverb

The MNM reverb is a hall/room model with predelay: 

; Predelay → reverb:
FX_send_bus
|
[delwrite~ mnm_pre 100]
[delread~ mnm_pre $predelay]    ; predelay: 0–100 ms
|
[rev3~ $size $verbTime $damp]
|                         |
wet L                   wet R
→ mixed with dry at $mix level

              FX bus → [delay/chorus/reverb] → [*~ fx_return] → main output

Summary

The Machinedrum provides per-track send levels to a global stereo delay (with HPF/LPF in the feedback path for tonal shaping) and a plate reverb. The Monomachine adds per-voice parametric EQ and distortion, a modulated stereo delay (chorus-delay), a global chorus/flanger, and a predelay+reverb. Both architectures use a send/return model — dry signal passes to the main output, and a send fraction goes into the shared effects bus. In Pd, implement with [delread~]/[delwrite~] for delays, [rev3~] for reverb, and [biquad~] for parametric EQ.
