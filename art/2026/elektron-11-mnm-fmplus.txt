Per-Operator Index Envelopes


In FM+, each operator can have its own index envelope ŌĆö meaning M1 and M2 can evolve
at different rates. A common FM brass configuration: M1 has a fast attack/decay (adds
brightness on the note onset), while M2 has a slow attack (adds complexity as the note
sustains):



; M1 index envelope: bright attack
[vline~]    ; 0 0, idx1_peak 5, idx1_peak*0.3 80 5    ; attack 5ms, decay to 30%

; M2 index envelope: slow modulation build
[vline~]    ; 0 0, 0 50, idx2_peak 400 50             ; builds over 400ms



This two-envelope approach (one fast, one slow) is the signature of FM brass and string
sounds. The fast envelope provides the attack transient; the slow envelope provides the
evolving shimmer.


Carrier Feedback in FM+


FM+ adds carrier self-feedback (FB C) in addition to VO's modulator feedback. The
carrier feeds back into its own frequency input through a one-sample delay:



; Carrier feedback:
carrier_output ŌåÆ [delread~ carr_fb 1/sr]
|
[*~ fb_c_scaled]
|
[+~ carrier_freq_modulated]    ; add to carrier's total frequency input
|
[osc~]                          ; carrier ŌĆö self-modulating

[delwrite~ carr_fb 1/sr] ŌåÉ carrier_output



Carrier feedback at moderate levels adds a sawtooth-like harmonic richness without
affecting the modulator's spectrum. At high levels it adds chaotic FM behaviour.


Detune


The DETUNE parameter slightly offsets operator frequencies against each other, creating
slow beating between operators. This is especially effective with parallel algorithm:
both modulators slightly detuned from the carrier produce a rich chorus-like motion
in the FM sidebands:



; Detune: offset M1 and M2 by +/- a small fraction of their base frequency
; detune_hz = carrier_freq ├Ś detune ├Ś 0.02    (up to 2% detuning)

freq_m1 = carrier_freq ├Ś ratio_1 + detune_hz * 0.5
freq_m2 = carrier_freq ├Ś ratio_2 - detune_hz * 0.5

Summary


FM+ extends VO with a third operator, three algorithm topologies (series, parallel,
hybrid), per-operator ADSR index envelopes, and carrier self-feedback. The algorithm
selection is the primary timbral structural choice: series produces "nested" FM
textures; parallel produces additive FM sideband clusters; hybrid combines both.
Per-operator envelopes enable the characteristic FM brass sound (fast M1 for bright
attack, slow M2 for sustained shimmer). Carrier feedback adds harmonic richness to the
output without involving the modulator chain.
