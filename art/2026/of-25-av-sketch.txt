openFrameworks - Part 25: Putting It Together - Audio-Visual Sketch
2026-06-01


Fourteen articles of building blocks converge here. A microphone captures live audio.
The FFT decomposes it into frequency bands - bass, mids, highs. A vertex shader deforms a 3D
grid mesh with those bands; a fragment shader colours it with a palette driven by amplitude.
An ofEasyCam lets you orbit the scene in real time. ofxGui sliders
control decay, mesh scale, and colour intensity without touching the code. Press 'f' for
fullscreen; press 's' to save a PNG. This is not a toy - it is a complete, deployable sketch
built from everything in the series.


What You Will Learn


  How all the series' building blocks compose into a single coherent application.
  The threading model: audio callback on its own thread, FFT on the main thread, GPU upload per frame.
  How to pass per-frequency-band data to a vertex shader via a 1D texture.
  Clean separation of concerns: audio, analysis, rendering, and UI in distinct methods.
  A retrospective on the openFrameworks series and pointers to where to go next.

    // --- Audio I/O ---
    ofSoundStream   soundStream;
    std::mutex      audioMutex;
    std::vector<float> audioBuffer;       // latest audio frame from callback

    // --- FFT ---
    static const int FFT_SIZE = 1024;
    std::vector<float> fftWindow;         // Hann window coefficients
    std::vector<float> fftMagnitudes;     // log-magnitude spectrum, FFT_SIZE/2 bins
    std::vector<float> smoothMag;         // exponentially smoothed magnitudes

    float bass   = 0.0f;   // 20-200 Hz energy
    float mids   = 0.0f;   // 200-2000 Hz energy
    float highs  = 0.0f;   // 2000-10000 Hz energy
    float rms    = 0.0f;   // overall amplitude

    ofTexture fftTex;

    // --- Grid mesh ---
    ofMesh    grid;
    int       gridCols = 64;
    int       gridRows = 64;
    float     gridSpacing = 14.0f;

    // --- Scene ---
    ofEasyCam cam;
    ofLight   sun;
    ofShader  avShader;

    // --- GUI ---
    ofxPanel  gui;
    ofParameter<float> pDecay      {"Decay",       0.88f, 0.5f,  0.99f};
    ofParameter<float> pMeshScale  {"Mesh Scale",  80.0f, 10.0f, 300.0f};
    ofParameter<float> pColorAmp   {"Colour Amp",  1.5f,  0.1f,  4.0f};
    ofParameter<float> pBrightness {"Brightness",  1.0f,  0.0f,  3.0f};
    ofParameter<bool>  pWireframe  {"Wireframe",   false};
    ofParameterGroup   params;

    bool guiVisible = true;

    // --- Stats ---
    float avgFps = 60.0f;
};


ofApp.cpp


#include "ofApp.h"
#include <algorithm>
#include <cmath>

// ---------- Lifecycle ----------

void ofApp::setup() {
    ofSetFrameRate(60);
    ofBackground(8, 8, 16);
    ofEnableDepthTest();
    ofSetWindowTitle("AV Sketch");

    // --- Audio ---
    audioBuffer.assign(FFT_SIZE, 0.0f);
    ofSoundStreamSettings ss;
    ss.setInListener(this);
    ss.sampleRate        = 44100;
    ss.numInputChannels  = 1;
    ss.numOutputChannels = 0;
    ss.bufferSize        = 512;
    soundStream.setup(ss);

    // --- FFT ---
    // Hann window
    fftWindow.resize(FFT_SIZE);
    for (int i = 0; i < FFT_SIZE; i++) {
        fftWindow[i] = 0.5f * (1.0f - std::cos(2.0f * M_PI * i / (FFT_SIZE - 1)));
    }
    fftMagnitudes.assign(FFT_SIZE / 2, 0.0f);
    smoothMag.assign(FFT_SIZE / 2, 0.0f);

    fftTex.allocate(FFT_SIZE / 2, 1, GL_R32F);

    // --- Mesh ---
    buildGrid();

    // --- Shader ---
    avShader.load("shaders/av");

    // --- Camera ---
    cam.setDistance(600);
    cam.setPosition(0, -180, 600);
    cam.lookAt(glm::vec3(0, 0, 0));

    // --- Light ---
    sun.setDirectional();
    sun.setOrientation(glm::vec3(35, -20, 0));
    sun.setDiffuseColor(ofColor(200, 210, 255));
    sun.setSpecularColor(ofColor(255, 255, 255));

    // --- GUI ---
    params.setName("AV Sketch");
    params.add(pDecay, pMeshScale, pColorAmp, pBrightness, pWireframe);
    gui.setup(params);
    gui.setPosition(10, 40);
}

void ofApp::exit() {
    soundStream.close();
}

void ofApp::update() {
    // Collect latest audio frame safely
    bool hasFrame = false;
    {
        std::lock_guard<std::mutex> lock(audioMutex);
        hasFrame = newAudioFrame;
        newAudioFrame = false;
    }

    if (hasFrame) {
        computeFFT();
        updateFFTTexture();
    }

    deformGrid();

    avgFps = avgFps * 0.95f + ofGetFrameRate() * 0.05f;
}

void ofApp::draw() {
    drawScene();
    if (guiVisible) drawHUD();
}

void ofApp::keyPressed(int key) {
    if (key == 'f' || key == 'F') ofToggleFullscreen();
    if (key == 'g' || key == 'G') guiVisible = !guiVisible;
    if (key == 's' || key == 'S') ofSaveScreen("screenshot_" + ofGetTimestampString() + ".png");
}

// ---------- Audio ----------

void ofApp::audioIn(ofSoundBuffer &buf) {
    std::lock_guard<std::mutex> lock(audioMutex);
    size_t n = std::min(buf.getNumFrames(), (size_t)FFT_SIZE);
    for (size_t i = 0; i < n; i++) {
        audioBuffer[i] = buf.getSample(i, 0);
    }
    // Compute RMS for overall amplitude
    float sum = 0.0f;
    for (size_t i = 0; i < n; i++) sum += audioBuffer[i] * audioBuffer[i];
    rms = std::sqrt(sum / n);
    newAudioFrame = true;
}

// ---------- FFT ----------

void ofApp::computeFFT() {
    // oF ships with a simple DFT/FFT via ofSoundUtils
    // We use ofSoundGetSpectrum() for brevity; in production use kiss_fft or pffft.
    // Here we integrate the audio buffer directly using a simplified DFT
    // to keep the example self-contained without external FFT libs.
    //
    // In a real project, replace this with:
    //   float* spec = ofSoundGetSpectrum(FFT_SIZE / 2);
    //   for (int i = 0; i < FFT_SIZE / 2; i++) fftMagnitudes[i] = spec[i];

    int N = FFT_SIZE;
    int halfN = N / 2;
    float decay = pDecay.get();

    // Simple DFT (for illustration - replace with FFT for production)
    // Using the audio buffer captured in audioIn()
    for (int k = 0; k < halfN; k++) {
        float re = 0.0f, im = 0.0f;
        for (int n = 0; n < N; n++) {
            float w = fftWindow[n] * audioBuffer[n];
            float theta = 2.0f * M_PI * k * n / N;
            re += w * std::cos(theta);
            im -= w * std::sin(theta);
        }
        float mag = std::sqrt(re * re + im * im) / (N * 0.5f);
        // Log-magnitude with floor
        fftMagnitudes[k] = 20.0f * std::log10(mag + 1e-6f);
        fftMagnitudes[k] = (fftMagnitudes[k] + 60.0f) / 60.0f;   // normalise -60..0 dBFS to 0..1
        fftMagnitudes[k] = ofClamp(fftMagnitudes[k], 0.0f, 1.0f);

        // Smooth
        if (fftMagnitudes[k] > smoothMag[k]) {
            smoothMag[k] = fftMagnitudes[k];          // instant attack
        } else {
            smoothMag[k] = smoothMag[k] * decay + fftMagnitudes[k] * (1.0f - decay);
        }
    }

    // Extract bands
    bass  = bandEnergy(0,  22);    // 0-22 bins ~= 0-950 Hz @ 44100 / 1024
    mids  = bandEnergy(22, 200);
    highs = bandEnergy(200, halfN - 1);
}

float ofApp::bandEnergy(int binLow, int binHigh) {
    float sum = 0.0f;
    int count = binHigh - binLow + 1;
    for (int i = binLow; i <= binHigh && i < (int)smoothMag.size(); i++) {
        sum += smoothMag[i];
    }
    return sum / count;
}

void ofApp::updateFFTTexture() {
    fftTex.loadData(smoothMag.data(), FFT_SIZE / 2, 1, GL_RED);
}

// ---------- Mesh ----------

void ofApp::buildGrid() {
    grid.clear();
    grid.setMode(OF_PRIMITIVE_TRIANGLES);

    float hw = (gridCols - 1) * gridSpacing * 0.5f;
    float hd = (gridRows - 1) * gridSpacing * 0.5f;

    for (int r = 0; r < gridRows; r++) {
        for (int c = 0; c < gridCols; c++) {
            float x = c * gridSpacing - hw;
            float z = r * gridSpacing - hd;
            grid.addVertex(glm::vec3(x, 0, z));
            grid.addNormal(glm::vec3(0, 1, 0));
            grid.addTexCoord(glm::vec2((float)c / (gridCols - 1),
                                       (float)r / (gridRows - 1)));
        }
    }
    for (int r = 0; r < gridRows - 1; r++) {
        for (int c = 0; c < gridCols - 1; c++) {
            int tl = r * gridCols + c;
            int tr = tl + 1;
            int bl = tl + gridCols;
            int br = bl + 1;
            grid.addIndex(tl); grid.addIndex(bl); grid.addIndex(tr);
            grid.addIndex(tr); grid.addIndex(bl); grid.addIndex(br);
        }
    }
}

void ofApp::deformGrid() {
    float t       = ofGetElapsedTimef();
    float scale   = pMeshScale.get();
    float hw = (gridCols - 1) * gridSpacing * 0.5f;
    float hd = (gridRows - 1) * gridSpacing * 0.5f;

    for (int r = 0; r < gridRows; r++) {
        for (int c = 0; c < gridCols; c++) {
            int idx = r * gridCols + c;
            float x = c * gridSpacing - hw;
            float z = r * gridSpacing - hd;

            // Perlin noise base + band-driven spikes
            float n   = ofNoise(c * 0.05f + t * 0.2f, r * 0.05f - t * 0.15f);
            float y   = (n - 0.5f) * scale;

            // Bass punches the grid up; highs add fine ripple
            float dist = std::sqrt(x * x + z * z);
            y += bass  * scale * 0.6f * std::exp(-dist * 0.002f);
            y += highs * scale * 0.2f * std::sin(dist * 0.1f - t * 4.0f);

            grid.setVertex(idx, glm::vec3(x, y, z));

            // Recompute normal via finite difference
            int ci = std::min(c + 1, gridCols - 1);
            int ri = std::min(r + 1, gridRows - 1);
            glm::vec3 right = grid.getVertex(r  * gridCols + ci);
            glm::vec3 below = grid.getVertex(ri * gridCols + c);
            glm::vec3 here  = grid.getVertex(idx);
            grid.setNormal(idx, glm::normalize(glm::cross(right - here, below - here)));
        }
    }
}

// ---------- Rendering ----------

void ofApp::drawScene() {
    sun.enable();
    ofEnableLighting();

    cam.begin();
        avShader.begin();
            avShader.setUniform1f("uTime",       ofGetElapsedTimef());
            avShader.setUniform1f("uBass",        bass);
            avShader.setUniform1f("uMids",        mids);
            avShader.setUniform1f("uHighs",       highs);
            avShader.setUniform1f("uRMS",         rms);
            avShader.setUniform1f("uColorAmp",    pColorAmp.get());
            avShader.setUniform1f("uBrightness",  pBrightness.get());
            avShader.setUniformTexture("uFFT",    fftTex, 1);

            if (pWireframe.get()) {
                grid.drawWireframe();
            } else {
                grid.draw();
            }
        avShader.end();
    cam.end();

    ofDisableLighting();
    sun.disable();
}

void ofApp::drawHUD() {
    ofDisableDepthTest();
    gui.draw();

    ofSetColor(200);
    ofDrawBitmapString("FPS: " + ofToString((int)avgFps) +
                       "  BASS: " + ofToString(bass, 2) +
                       "  MIDS: " + ofToString(mids, 2) +
                       "  HIGHS: " + ofToString(highs, 2) +
                       "\n[G] toggle GUI  [F] fullscreen  [S] screenshot",
                       10, 20);
    ofEnableDepthTest();
}

// data/shaders/av.vert
#version 120

uniform float uTime;
uniform float uBass;
uniform float uMids;
uniform float uHighs;
uniform sampler2D uFFT;

varying vec2 vTexCoord;
varying float vY;        // normalised height for colouring

void main() {
    vTexCoord = gl_MultiTexCoord0.xy;
    vY        = gl_Vertex.y;         // after CPU deformation

    gl_Position = ftransform();
}


Fragment Shader: av.frag


// data/shaders/av.frag
#version 120

uniform float uTime;
uniform float uBass;
uniform float uMids;
uniform float uHighs;
uniform float uRMS;
uniform float uColorAmp;
uniform float uBrightness;
uniform sampler2D uFFT;

varying vec2 vTexCoord;
varying float vY;

vec3 palette(float t) {
    // Cosine colour palette (Inigo Quilez technique)
    vec3 a = vec3(0.5, 0.5, 0.5);
    vec3 b = vec3(0.5, 0.5, 0.5);
    vec3 c = vec3(1.0, 1.0, 0.5);
    vec3 d = vec3(0.0, 0.10, 0.20);
    return a + b * cos(6.28318 * (c * t + d));
}

void main() {
    // Sample FFT at the U coordinate of this fragment
    float bin = vTexCoord.x;
    float mag = texture2D(uFFT, vec2(bin, 0.5)).r;

    // Height-based colour: low vertices are dark/cool, high vertices are bright/warm
    float t = (vY / 200.0) * 0.5 + 0.5;   // normalise height to ~0..1
    t += uBass  * 0.3 * uColorAmp;
    t += mag    * 0.2 * uColorAmp;
    t += uTime  * 0.04;

    vec3 col = palette(t);

    // Brighten with overall amplitude
    col *= uBrightness * (0.6 + uRMS * 2.0);

    // Rim: faint cyan edge on frequency peaks
    col += vec3(0.0, 0.3, 0.4) * mag * uMids * 2.0;

    col = clamp(col, 0.0, 1.0);

    gl_FragColor = vec4(col, 1.0);
}


How the Parts Connect


Audio thread (Part 9): audioIn() copies the microphone buffer and
computes RMS under a mutex. The flag newAudioFrame signals the main thread
that fresh data is available.



FFT (Part 11): computeFFT() runs on the main thread in
update(), windowing the buffer with a Hann window and computing log-magnitude
spectrum. Band energies (bass, mids, highs) are extracted as averages over frequency ranges.



1D texture (Part 11, Part 5): updateFFTTexture() uploads
smoothMag as a 512×1 GL_R32F texture, passed to the fragment shader
as uFFT.



3D mesh (Part 12): buildGrid() builds a 64×64 indexed grid once in
setup(). deformGrid() runs in update(), driving
vertex heights with Perlin noise modulated by bass and highs.



Shader visuals (Part 11): The fragment shader applies a cosine colour palette
(Inigo Quilez's technique) parameterised by vertex height, FFT bin magnitude, and time.
Band energies modulate the palette offset and brightness.



Camera (Part 12): ofEasyCam provides mouse-driven orbit. All 3D drawing
is bracketed by cam.begin() / cam.end(); the 2D HUD is drawn
outside.



GUI (Part 8): ofxGui exposes five live parameters: smoothing decay,
mesh deformation scale, colour amplitude, brightness, and a wireframe toggle. All parameters
are read inside update() or drawScene(); no restart is needed to
change them.


Series Retrospective


The openFrameworks series started with the simplest possible question - what is oF, and why
does it exist - and ended here, with a sketch that simultaneously processes audio, analyses
frequency content, deforms a 3D mesh, runs a custom shader, and provides a live-tweakable
GUI. Along the way the series covered:



  Parts 1-7: the oF architecture, 2D drawing, events, timing, images, video, and typography.
  Parts 8-11: GUI, audio I/O, OSC, and shaders - the interactive, generative core.
  Parts 12-14: 3D geometry, the addon system, and YAML configuration.
  Parts 15-20: platform depth - Linux, Windows, macOS, Android, and portable coding patterns.
  Parts 21-24: lower-level graphics - SDL2, OpenGL core profile, framebuffers, and compute shaders.
  Part 25: synthesis.

Summary


This sketch demonstrates the composition pattern that underpins all creative-coding work:
audio input isolated on its own thread, analysis on the main thread, GPU upload of analysis
results, geometry deformation driven by audio, shader colouring driven by spectral data,
interactive 3D navigation, and live parameter control - all wired together with clear data
flow and proper thread safety. The individual pieces are the articles in this series. The
composition is yours to extend.
